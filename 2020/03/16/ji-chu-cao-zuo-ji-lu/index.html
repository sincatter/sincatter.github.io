BIOS设置

联想E470 启动BIOS为开机后Enter->F1，如果只是指定磁盘启动顺序则只需要F12。
特别的，Win10如果想要修改BIOS相关内容需要，取消设置里的快速启动，并(shif+点击关机)->疑难解答->高级选项->UEFI固件设置。

虚拟环境安装scrapy

  • Ubuntu 18.0.4 LTS使用update-alterantive配置python2/3共存
    # 添加python版本，并指定优先级
sudo update-alternatives --install /usr/bin/python python /usr/local/lib/python3.6 100
sudo update-alternatives --install /usr/bin/python python /usr/local/lib/python2.7 200
# 选择指定版本
sudo update-alternatives --config python
  • 创建虚拟环境
    # 安装virtualenv
pip3 install -i https://pypi.douban.com/simple/ virtualenv
# 创建虚拟环境目录，与实际代码位置可以不同
virtualenv venv_scrapy
## 切换虚拟目录
cd venv_scrapy && source ./bin/activate
## 安装scrapy
pip3 install -i https://pypi.douban.com/simple/ scrapy
## 创建scrapy项目
cd ~/Code && scrapy startproject scrapy_prj
## 退出虚拟环境
deactivate
