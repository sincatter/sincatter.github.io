## virtualbox

> Windows/Linux/Mac
> 当前OS: Ubuntu 18.04 LTS

* 主程序，安装时仓库版本为5.2.34
```
sudo apt-get install virtualbox
```
* 扩展包（提供USB扩展支持）
[https://www.virtualbox.org/wiki/Download_Old_Builds_5_2(https://www.virtualbox.org/wiki/Download_Old_Builds_5_2)(历史版本，版本与主程序对应)
* 官方Download地址
