# 单向链表反转
## 直接递归

> 时间复杂度$O(n)$，空间复杂度$O(n)$   

简单递归输出就好，适合小数据量。  
```c
void reverse(head){
	if(head.next == null){
		print(head);
		return;
	}
	reverse(head.next);
	print(head);
}
```
## 栈存储

> 时间复杂度$O(n)$，空间复杂度$O(n)$  

分为两种:存储元素地址和存储值，其中地址可以构建出一个新的逆序列表。以存储地址为例：  
```c
void reverse(head){
	Node cur = head;
	while(cur != null){
		stack.push(cur);
		cur = cur.next;
	}
	while(!stack.isEmpty){
		print(stack.pull());
	}
}
```

## 反转链表结构

> 时间复杂度$O(n)$，空间复杂度0  

这种方法不需要额为的空间，在遍历时调整元素的指向关系。  
![单向链表反转](https://raw.githubusercontent.com/sincatter/picgos/imgs/imgs20191006222506.png)
```c
void reverse(head){
	Node* cur = head.next;
	Node* pre = head;
    pre.next = nullptr;
	Node* tmp = nullptr;
	while(cur != null){
		tmp = cur.next;
	    cur.next = pre;
		pre = cur;
		cur = tmp;
	}
}
```
## LeetCode例子
### 题目 - 92. Reverse Linked List II
Reverse a linked list from position *m* to *n*. Do it in one-pass.  
**Note:** 1 ≤ *m* ≤ *n* ≤ length of list.  
**Example:**  
> **Input:** 1->2->3->4->5->NULL, *m* = 2, *n* = 4
> **Output:** 1->4->3->2->5->NULL

反转从位置 m 到 n 的链表。请使用一趟扫描完成反转。  
**说明:**  
1 ≤ m ≤ n ≤ 链表长度。  
**示例:**  
> **输入:** 1->2->3->4->5->NULL, m = 2, n = 4
> **输出:** 1->4->3->2->5->NULL

### 解答
```c
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
public:
    ListNode* reverseBetween(ListNode* head, int m, int n) {
        ListNode *mNode = new ListNode(0);
        ListNode *resNode = nullptr;
        ListNode *pre = mNode;
        mNode->next = head;
        for(int i = 0;i < m - 1; i++){
            pre = pre->next;
        }
        ListNode *cur = pre->next;
        ListNode *tmp = nullptr;
        for (int i = m; i < n;i++){
            tmp = pre->next;
            pre->next = cur->next;
            cur->next = pre->next->next;
            pre->next->next = tmp;
        }
        resNode = mNode->next;
        delete(mNode);
        return resNode;
    }
};
```
